### 总体概览
- 首页: 各个产品数据入口
- 测试源数据
- 用例详情（依赖和最新结果）
- 用例依赖关系表
- 用例最新测试结果表
- 公用变量配置表
- 全部用例测试详情表
- 单个用例测试详情表
- 测试报告表

### 测试原理
- 数据与代码分离：
	- 代码（common.py-测试逻辑/main.py-路由信息/templates-jinja2模板）
	- 数据（Config-API依赖、变量定义、各产品信息/CSV-由swagger数据解析后的数据/HTML-swagger数据/LOG-日志和测试报告）
- 无依赖用例跑多线程/有依赖用例跑单线程
- 多值变量排列组合遍历测试
- 支持全部测试（依赖前置变量测试单次）和单个用例测试（遍历全部变量值）
